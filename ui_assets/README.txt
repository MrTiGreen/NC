NightClub UI Asset Pack

Содержимое:
- png/: прозрачные PNG ассеты интерфейса
- svg/: базовые масштабируемые SVG рамки/слоты/кнопки
- asset_preview_atlas.png: превью всех главных ассетов
- manifest.json: размеры, названия и палитра

В наборе нет персонажей и предметов. Только интерфейс, пустые слоты и заглушки.

Рекомендация для Codex:
1) Сначала собрать экран на CSS grid/flex по этим размерам.
2) Использовать PNG как background-image для прототипа.
3) Позже заменить ключевые элементы на SVG/CSS, чтобы интерфейс лучше масштабировался.
